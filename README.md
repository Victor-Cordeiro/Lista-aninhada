## 📝 Sobre o Projeto
Este repositório contém a implementação de uma **Lista Aninhada** (Nested List) desenvolvida em C++. O projeto foi criado com o objetivo de demonstrar o funcionamento, a manipulação e o gerenciamento de memória dessa estrutura de dados clássica.

## ✨ Funcionalidades Implementadas
* **Inserção:** Capacidade de adicionar novos elementos (ou sublistas) na estrutura.
* **Remoção:** Exclusão de elementos específicos ou sublistas.
* **Busca:** Pesquisa por valores específicos dentro das listas e sublistas.
* **Exibição:** Impressão formatada no console de toda a estrutura aninhada.
* **Gerenciamento de Memória:** Alocação e liberação correta de ponteiros (prevenção de memory leaks).
